## Présentation

Kolr est la boîte à outils couleur complète pour les designers, développeurs et créatifs.

## Fonctionnalités

- **Générateur de palettes** : créez des palettes de couleurs harmonieuses instantanément
- **Accessibilité WCAG** : vérifiez les ratios de contraste et la conformité
- **Simulation daltonisme** : prévisualisez vos designs tels que les daltoniens les voient
- **Conversion de formats** : convertissez entre HEX, RGB, HSL, OKLCH et plus
- **Générateur de thèmes sombres** : générez automatiquement des variantes dark mode

## Disponible sur

- **Web** : site vitrine & application web en Next.js
- **iOS** : application native en React Native & Expo
- **Extension Chrome** : accès rapide depuis la barre d'outils du navigateur
- **Add-on Firefox** : également disponible sur Firefox
- **Plugin Figma** : utilisez Kolr directement dans Figma
