## Présentation

Re:mind est une application de rappels intelligents conçue pour simplifier le quotidien. Elle gère les rappels classiques, le suivi de médicaments, les alertes de péage Flux Libre, et bien plus.

## Fonctionnalités

- **Rappels classiques** : programmez des rappels ponctuels ou récurrents
- **Suivi médicaments** : ne manquez plus jamais une prise grâce aux notifications intelligentes
- **Péages Flux Libre** : soyez rappelé de payer vos péages avant la date limite
- **Multi-plateforme** : application Android native + site web vitrine

## Stack technique

Application mobile développée avec React Native et Expo. Site vitrine construit avec Next.js et TypeScript.
